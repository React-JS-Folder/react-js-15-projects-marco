There are 2 ways to style a React component 
- Inline  (You need to capitalize the properties name)
- External style sheet (You need to add the link to the css file in the component)
##################################
Inline
 <h4 style={{ 
   color: '#ccc',
   backgroundColor: 'grey',
   maxWidth: '200px'
 }}

##################################
 External

 Book.js
 import './cssFile.css';
 
 function Book(){
   return(
     <h1>Hello Book</h1>
   )
 }
